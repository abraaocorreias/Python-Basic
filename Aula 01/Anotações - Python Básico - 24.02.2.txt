O que é um linguagem de programação?
	Uma linguagem é um conjunto de regras e instruções que permite desenvolvedores a criarem softwares.

Python:
	1991 - Guido Van Rossum
	Simples de Aprender e Ler;
	Multiplataforma;
	Usada para scripts simples quanto para sistemas complexos;
	Linguagem principal para desenvolvimentos de IA;
	possui muitas bibliotecas prontas, facilitando qualquer projeto.

Aplicações:
	Desenvolvimento WEB
		Bibliotecas: Django, Flask, FastAPI
	Ciência de Dados e Inteligência artificial
		Bibliotecas: Pandas, NumPy, TensorFlow, Scikit-Learn
	Automação de Tarefas
		Exemplo: Automatizar o backup de arquivos em empresas, bots que coletam preços em sites de e-comerce.
	Desenvolvimento de Jogos
		Bibliotecas: Pygame
